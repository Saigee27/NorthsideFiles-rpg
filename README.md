# NorthsideFiles RPG

NorthsideFiles RPG is a text-based role-playing game built in C++. The project was created to strengthen practical understanding of object-oriented programming through the implementation of core RPG mechanics.

## Implemented Systems

* Character class hierarchy
* Turn-based combat
* Health and damage system
* Distinct attack behavior for different character types

## Technical Concepts

* Encapsulation
* Inheritance
* Polymorphism
* Pure Virtual Functions
* Constructors
* Random Number Generation

## Status

Actively under development. The current version is playable through the command line and continues to evolve as new mechanics are implemented.
